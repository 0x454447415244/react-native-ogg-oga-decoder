# react-native-ogg-oga-decoder
React Native OGG OGA Decoder
## Installation

```sh
npm install react-native-ogg-oga-decoder
```

## Usage

```js
import { multiply } from "react-native-ogg-oga-decoder";

// ...

const result = await multiply(3, 7);
```
